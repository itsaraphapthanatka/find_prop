# HOB — สรุปฟีเจอร์ทั้งระบบ

> ระบบจัดการทรัพย์ให้เช่า/ขายสำหรับทีมนายหน้า — โกดัง โรงงาน โชว์รูม ออฟฟิศ ครัวกลาง
> ใช้แทนแอป AppSheet เดิมทั้งหมด · อัปเดตล่าสุด 18 ก.ค. 2026

**Stack:** React + TypeScript + Vite · Supabase (PostgreSQL / Auth / Storage / RLS) · Leaflet · PWA · Vercel · LLM self-hosted (vLLM)

---

## 1. รายการทรัพย์ (หน้าหลัก)

- ค้นหาจากทุกฟิลด์ + ตัวกรอง: ประเภททรัพย์, เช่า/ขาย, จังหวัด, ช่วงราคา
- มือถือแสดงเป็นการ์ดอ่านง่าย, จอใหญ่เป็นตาราง — responsive ทุกขนาดจอ
- เปิดรายละเอียดครบทุกฟิลด์ พร้อมรูปภาพ แผนที่ และปุ่มลัด โทร / SMS / LINE / เปิด Google Maps
- รองรับ deep-link ตัวกรอง (เช่น จาก Dashboard จิ้มกราฟ → เปิดรายการพร้อมกรองให้เลย)

## 2. ฟอร์มเพิ่ม/แก้ไขทรัพย์

- ครบ ~50 ฟิลด์ตามโครงสร้างข้อมูลต้นแบบ (ราคา พื้นที่ ไฟฟ้า พื้นรับน้ำหนัก โซนสี เจ้าของ ฯลฯ)
- Dropdown แบบ "เลือกหรือพิมพ์เพิ่ม" และ multi-select (คุณสมบัติเด่น / การใช้งาน / โซน)
- อัปโหลดรูปหลายรูปขึ้น Supabase Storage
- ปักพิกัดจากแผนที่ในฟอร์มได้โดยตรง

## 3. บันทึกทรัพย์ด้วยเสียง 🎤 + AI กรอกฟอร์ม

- กดพูดหน้างาน เล่ารายละเอียดทรัพย์รวดเดียว → AI แกะเป็นฟิลด์แล้วกรอกฟอร์มให้อัตโนมัติ
- เข้าใจตัวเลขที่พูดเป็นคำไทย เช่น "ค่าเช่าแปดหมื่นห้า" → 85,000
- แยกข้อมูลถูกช่อง: จุดเด่นเข้า "คุณสมบัติ" ไม่ปนกับหมายเหตุ, ไม่แต่งข้อมูลที่ไม่ได้พูด
- ถอดเสียงด้วย Web Speech API ในเบราว์เซอร์ (ฟรี, Chrome/Android แม่นสุด) — เบราว์เซอร์ที่ไม่รองรับจะซ่อนปุ่มให้เอง

## 4. แผนที่รวมทรัพย์

- ปักหมุดทรัพย์ทั้งหมดที่มีพิกัดบนแผนที่เดียว (Leaflet/OpenStreetMap)
- Popup สรุปย่อ + กดเปิดรายละเอียดเต็ม
- หมุดร่างสีม่วงสำหรับกำหนดตำแหน่งทรัพย์ใหม่

## 5. แผนเยี่ยมชม (จัดรูทพาลูกค้าดูทรัพย์)

- สร้างแผนต่อลูกค้า เลือกทรัพย์เป็นจุดแวะ จัดลำดับก่อน-หลังได้
- เปิดเส้นทางใน Google Maps ทุกจุดแวะในคลิกเดียว — เริ่มนำทางจากตำแหน่งปัจจุบัน
- **AI จับคู่ requirement ใหม่:** ลูกค้าเปลี่ยนความต้องการกลางทาง (พิมพ์หรือพูดใส่) → AI สแกนทรัพย์ทั้งระบบ แนะนำตัวที่ตรง พร้อมชี้ว่าจุดแวะเดิมตัวไหนไม่ตรงแล้ว → กดเพิ่มเข้ารูทได้ทันที

## 6. ชอร์ตลิสต์เปรียบเทียบ (เสนอลูกค้า)

- เลือกทรัพย์ 2–4 รายการ เทียบสเปกเป็นตารางเรียงข้าง
- AI เขียนบทวิเคราะห์ จุดเด่น / ข้อควรพิจารณา / คำแนะนำ ประกอบการตัดสินใจ
- พิมพ์ / บันทึก PDF เป็นเอกสารเปรียบเทียบหัวกระดาษ HOB ส่งลูกค้าได้ทันที
- เปิดจากหน้ารายการ หรือสั่งผ่านผู้ช่วย AI ก็ได้

## 7. นำเข้าข้อมูลจาก Excel / CSV

- อัปโหลดไฟล์ (เช่น export จาก AppSheet / Google Sheets เดิม) แบบ 3 ขั้นตอน: อัปโหลด → ตรวจ/จับคู่คอลัมน์ → นำเข้า
- จับคู่หัวคอลัมน์กับฟิลด์อัตโนมัติ (รู้จักชื่อไทยหลายแบบ), แก้การจับคู่เองได้
- รองรับวันที่ พ.ศ. / วันที่แบบ Excel / เลขมีคอมมา — แปลงให้ถูกต้อง
- ตรวจรหัสทรัพย์ซ้ำ เลือก "ข้าม" หรือ "อัปเดตทับ" ได้ + มีเทมเพลต CSV ให้ดาวน์โหลด

## 8. Dashboard สรุปภาพรวม

- ตัวเลขพอร์ต: มูลค่าเช่า/ขายรวม, ทรัพย์ใหม่เดือนนี้ + sparkline ย้อนหลัง 6 เดือน
- นัดเยี่ยมชมที่ใกล้ถึง พร้อมปุ่มนำทาง
- สัดส่วนประเภททรัพย์ / เช่า-ขาย (แตะกราฟ → เปิดหน้ารายการพร้อมตัวกรอง)
- ทำเลยอดนิยม (Top อำเภอ)
- ค่าเช่าเฉลี่ยต่อ ตร.ม. แยกตามประเภท + ชี้ทรัพย์ที่ถูก/แพงกว่าค่าเฉลี่ยกลุ่มอย่างมีนัย
- สุขภาพข้อมูล: ทรัพย์ที่ยังไม่มีรูป / พิกัด / ราคา / เบอร์ติดต่อ / ข้อมูลค้างเก่าเกิน 90 วัน
- **AI วิเคราะห์พอร์ตรายวัน:** จุดแข็งของพอร์ต, demand ที่ยังไม่มีของ, ทรัพย์ที่ควรเร่งนำเสนอ (แคชวันละครั้ง)

## 9. ผู้ช่วย AI (แชทลอยทุกหน้า) ✨

- ถามข้อมูลทรัพย์/แผนเยี่ยมชมของทีมเองเป็นภาษาพูด เช่น "โกดังแถวบางพลีงบไม่เกินแสนมีไหม" → ตอบจากข้อมูลจริงพร้อมการ์ดทรัพย์กดดูต่อได้ ไม่มีข้อมูลก็บอกตรงๆ ไม่มโน
- สั่งงานผ่านแชทได้: เพิ่ม/ถอดทรัพย์จากรูท, สร้างแผนเยี่ยมชม, เปิดหน้าเปรียบเทียบ — ทุกคำสั่งมีปุ่มยืนยันก่อนทำจริง
- พูดถามด้วยไมค์ + โหมดอ่านคำตอบออกเสียง (เหมาะตอนขับรถ)
- จำบทสนทนาไว้ในเครื่อง เปิดต่อจากเดิมได้

## 10. ทีมและสิทธิ์การใช้งาน

- ล็อกอินด้วยอีเมล (Supabase Auth) — ข้อมูลแยกตามองค์กรด้วย Row Level Security ทั้งระบบ
- **หน้าทีม (แอดมินองค์กร):** เพิ่มสมาชิกลูกทีม, ดูรายชื่อ/บทบาท, เปลี่ยนชื่อองค์กร
- **หน้า Super Admin (เจ้าของระบบ):** เห็นทุกองค์กร จำนวนสมาชิก/ทรัพย์, ตั้งแพ็กเกจ free/pro/enterprise และวันหมดอายุ
- **ประวัติการใช้งาน (เฉพาะแอดมิน):** บันทึกอัตโนมัติว่าใครทำอะไรเมื่อไหร่ — เพิ่ม/แก้/ลบทรัพย์, แผนเยี่ยมชม, นำเข้าไฟล์, การใช้ AI — กรองตามประเภท/ค้นหาชื่อคนได้ ล็อกด้วย RLS: สมาชิกทั่วไปเปิดดูไม่ได้ และแก้/ลบย้อนหลังไม่ได้ (audit trail ถาวร)

## หน้าแรกสาธารณะสำหรับลูกค้า (Landing Page)

- คนทั่วไปเปิดเว็บ → เห็นหน้าแนะนำบริการ: ประเภททรัพย์ที่ให้บริการ, จุดเด่น, ขั้นตอนใช้บริการ 3 ขั้น, ช่องทางติดต่อ โทร/LINE/อีเมล
- ทีมงานกด "เข้าสู่ระบบทีมงาน" เพื่อเข้าแอป — คนที่ล็อกอินค้างอยู่เข้าแอปตรงเหมือนเดิม
- แก้เบอร์โทร/LINE/อีเมลได้จุดเดียวที่ค่าคงที่ `CONTACT` ใน `src/pages/LandingPage.tsx`

## ติดตั้งเป็นแอปมือถือ (PWA)

- เพิ่มลงหน้าจอโฮมได้ทั้ง Android/iOS ใช้เหมือนแอปจริง อัปเดตตัวเองอัตโนมัติ
- ออกแบบ mobile-first: เมนูล่างแบบแอป, ตารางกลายเป็นการ์ด, ปุ่ม/ฟอนต์ขนาดเหมาะกับนิ้ว

## สถาปัตยกรรมและความปลอดภัย

- **AI ทุกฟีเจอร์เรียกผ่าน proxy ฝั่งเซิร์ฟเวอร์** (`api/ai.js` บน Vercel) — คีย์ LLM อยู่ใน environment variable ฝั่งเซิร์ฟเวอร์เท่านั้น ไม่มีทางหลุดจากหน้าเว็บ และตรวจว่าเป็นผู้ใช้ที่ล็อกอินจริงก่อนทุกครั้ง
- **LLM รันบนเครื่องของเราเอง** (gemma-4-12b บน vLLM, GPU RTX 5060 Ti) — ข้อมูลทรัพย์ไม่ออกไปเซิร์ฟเวอร์ AI ภายนอก และไม่มีค่าใช้จ่ายรายเดือน
- ก่อนส่งให้ AI ระบบคัดเฉพาะทรัพย์ที่เกี่ยวข้องกับคำถาม (relevance filter) — ตอบแม่นขึ้นและเร็วขึ้น
- Row Level Security ทุกตาราง: แต่ละองค์กรเห็นเฉพาะข้อมูลตัวเอง
