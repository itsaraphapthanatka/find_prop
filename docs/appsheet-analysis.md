# วิเคราะห์แอป AppSheet "WUT Demo" (ต้นแบบสำหรับ find_prop)

> สำรวจเมื่อ 2026-07-17 ผ่านหน้าแอปจริง (ต้องล็อกอิน Google)
> URL: https://www.appsheet.com/start/4894e65b-23f2-4fc2-8f33-0e5b926d634b

## ภาพรวม

แอปฐานข้อมูล**อสังหาริมทรัพย์ให้เช่า/ขาย** ประเภทโกดัง โรงงาน โชว์รูม ออฟฟิศ ครัวกลาง
มีตารางข้อมูลหลักตารางเดียว ("Wut Data") ข้อมูลตัวอย่าง 5 รายการ: `1`, `JKP01`, `JKP280`, `JKP373`, `JKP377`
พื้นที่ทรัพย์อยู่โซนกรุงเทพฯ–สมุทรปราการ

## Views

| View | ประเภท | รายละเอียด |
|---|---|---|
| **Map** | แผนที่ | ปักหมุดทรัพย์จากคอลัมน์ `เลขพิกัด` (LatLong), มีปุ่ม + Add |
| **Form** | ฟอร์ม | เพิ่ม/แก้ไขข้อมูลทรัพย์ (ฟิลด์ทั้งหมดด้านล่าง) |
| **Display From** | รายการ + Detail | list แสดง `ลำดับที่` + `วันที่`, คลิกเปิด detail ด้านขวา |
| About / Feedback | หน้า static | — |

### Action ต่อรายการ (ใน list และ detail)
- 📞 Phone call / 💬 Text message → จาก `เบอร์โทรติดต่อ`
- 🔗 Open Url (แผนที่) → เปิดลิงก์ Google Maps จากคอลัมน์ `แผนที่`
- 📍 View Map (เลขพิกัด) → ไปยังหมุดใน Map view
- ✏️ Edit / 🗑️ Delete

## โครงสร้างคอลัมน์ (ตาราง Wut Data)

`*` = required ในฟอร์ม | ประเภทข้อมูลตีความจาก UI ของฟอร์ม

### ข้อมูลทั่วไป
| คอลัมน์ | ชนิด | หมายเหตุ |
|---|---|---|
| วันที่ * | Date | default = วันนี้ |
| ลำดับที่ * | Text (รหัสทรัพย์) | เช่น JKP01, auto-suggest เลขถัดไป |
| รูป * | Image | ถ่าย/อัปโหลด |
| PIC * | Enum | default = ชื่อผู้ใช้ (เช่น Wut, Jacky) |

### ผู้ให้เช่า
| คอลัมน์ | ชนิด | ตัวเลือกที่พบ |

| สถานะ_ผู้ให้เช่า | Enum | บุคคล, (บริษัท?) |
| ชื่อบริษัท_ผู้ให้เช่า | Text | |
| ชื่อผู้ให้เช่า * | Text | |
| เบอร์โทรติดต่อ * | Phone | มี action โทร/SMS |
| เลขโฉนด | Text | |

### ประเภททรัพย์และทำเล

## ตัวอย่างข้อมูลจริง (JKP01)

โชว์รูมให้เช่า ต.เทพารักษ์ อ.เมืองสมุทรปราการ จ.สมุทรปราการ | ผู้ให้เช่า: บุคคล (K. เบนซ์, 088-888-8888) | อาคาร 1,457 ตร.ม. (23.5 x 62) + ออฟฟิศชั้น 1 = 100 → รวม 1,557 ตร.ม. | ค่าเช่า 350,000 บ./เดือน (240 บ./ตร.ม.) | ภาษีรวมแล้ว, ส่วนกลาง 5,000 | ประตู 1 บาน 4.8 x 3.9 ม., สูง 12 ม., พื้นรับ 3 ตัน, ไฟ 3 Phase 15/45 amp | สัญญา 3 ปี ประกัน 3 เดือน ล่วงหน้า 1 เดือน
