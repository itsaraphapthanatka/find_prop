# HUP — คู่มือการใช้งาน

> สำหรับทีมนายหน้า/เจ้าหน้าที่ที่ใช้ระบบทุกวัน · อัปเดต 20 ก.ค. 2026
> เปิดหน้าเว็บได้ที่ **hob-alpha.vercel.app/guide.html** (พิมพ์เป็น PDF ได้)

---

## เริ่มต้นใช้งาน

### เข้าสู่ระบบ
1. เปิดแอป HUP หรือเว็บ **hob-alpha.vercel.app** แล้วกด **เข้าสู่ระบบ**
2. ใส่อีเมลและรหัสผ่านที่แอดมินทีมสร้างให้ → กด **เข้าสู่ระบบ**
3. ยังไม่มีบัญชี? ติดต่อแอดมินทีมให้เพิ่มคุณเข้าระบบ (สมัครเองไม่ได้ — กันคนนอก)

### เมนูหลัก (แถบล่างของจอ)
| เมนู | ใช้ทำอะไร |
|---|---|
| **แผนที่** | ดูทรัพย์ทั้งหมดบนแผนที่ |
| **ฟอร์ม** | เพิ่มทรัพย์ใหม่ |
| **รายการ** | ค้นหา/กรอง/ดูรายการทรัพย์ |
| **สรุป** | ภาพรวมพอร์ต + AI วิเคราะห์ |
| **แผนเยี่ยม** | จัดรูทพาลูกค้าดูทรัพย์ |

ปุ่ม **✨ ลอยมุมขวาล่าง** = ผู้ช่วย AI (เรียกได้ทุกหน้า)

---

## 1. เพิ่มทรัพย์ใหม่ — มี 3 วิธี

### วิธี A — พิมพ์กรอกฟอร์มเอง
1. กด **ฟอร์ม** (หรือปุ่ม **+ เพิ่มทรัพย์** ที่หน้ารายการ/แผนที่)
2. กรอกข้อมูล: วันที่บันทึก, รหัสทรัพย์, ประเภท (โรงงาน/โชว์รูม/โกดัง/ออฟฟิศ/ครัวกลาง), เช่า/ขาย, ทำเล, ราคา ฯลฯ
3. ช่องที่มี * คือช่องบังคับ
4. กด **บันทึก**

### วิธี B — พูดให้ AI กรอกให้ 🎤 (เร็วที่สุดหน้างาน)
1. ที่หน้าฟอร์ม กดปุ่ม **พูด**
2. เล่ารายละเอียดทรัพย์รวดเดียวเป็นภาษาพูด เช่น
   *"โกดังให้เช่าแถวบางพลี พื้นที่ห้าร้อยตร.ม. ค่าเช่าแปดหมื่นห้า สูงแปดเมตร มีออฟฟิศในตัว"*
3. AI แกะเป็นฟิลด์แล้วกรอกให้อัตโนมัติ — **ตรวจความถูกต้องก่อนกดบันทึกทุกครั้ง**
4. พูดเป็นตัวเลขไทยได้ เช่น "แปดหมื่นห้า" → 85,000

> 💡 ปุ่มพูดแสดงบนเบราว์เซอร์ Chrome (คอม/Android) ถ้าไม่เห็นปุ่ม = อุปกรณ์นั้นไม่รองรับ ให้พิมพ์แทน

### วิธี C — นำเข้าจาก Excel/CSV (ย้ายข้อมูลเก่าทีเดียว)
1. เข้าหน้า **นำเข้าข้อมูล**
2. อัปโหลดไฟล์ → ระบบจับคู่คอลัมน์ให้อัตโนมัติ (แก้เองได้) → กดนำเข้า
3. ถ้ารหัสซ้ำ เลือก "ข้าม" หรือ "อัปเดตทับ" · มีเทมเพลต CSV ให้ดาวน์โหลด

### แนบรูปทรัพย์
- **บนแอปมือถือ:** กด **ถ่ายรูปด้วยกล้อง** ถ่ายหน้างานได้เลย (ครั้งแรกกดอนุญาตใช้กล้อง)
- **บนคอม/เว็บ:** กดกรอบรูป → เลือกไฟล์

### ปักตำแหน่งทรัพย์
- ในฟอร์มมีแผนที่เล็ก แตะเพื่อปักพิกัด หรือปักจากหน้าแผนที่ (ดูข้อ 3)

---

## 2. ค้นหาและกรองทรัพย์

1. กด **รายการ**
2. พิมพ์ในช่องค้นหา (ค้นได้จากรหัส ทำเล ประเภท ฯลฯ)
3. กรองเพิ่ม: ประเภท · เช่า/ขาย · จังหวัด · ช่วงราคา
4. แตะการ์ด/แถวทรัพย์เพื่อดูรายละเอียดเต็ม + ปุ่มลัด **โทร / SMS / LINE / เปิด Google Maps**

---

## 3. ดูทรัพย์บนแผนที่

1. กด **แผนที่** — เห็นหมุดทรัพย์ทั้งหมดที่มีพิกัด
2. **สีหมุด = ประเภททรัพย์** ดูคำอธิบายสีที่แถบล่างแผนที่ (legend)
3. แตะชิพประเภทที่ legend เพื่อ **ดูเฉพาะประเภทนั้น** · แตะ **ทั้งหมด** เพื่อกลับ
4. ปุ่ม **⊙ ตำแหน่งฉัน** (มุมล่างขวา) กระโดดไปจุดที่คุณอยู่ (ครั้งแรกกดอนุญาตตำแหน่ง)
5. เพิ่มทรัพย์จากแผนที่: กด **วางหมุดเพิ่มทรัพย์** → แตะตำแหน่งบนแผนที่ → กรอกฟอร์มต่อ

---

## 4. จัดแผนพาลูกค้าดูทรัพย์ (แผนเยี่ยมชม)

1. กด **แผนเยี่ยม** → **สร้างแผน**
2. ใส่ชื่อลูกค้า/ความต้องการ แล้วเลือกทรัพย์เป็นจุดแวะ จัดลำดับก่อน-หลังได้
3. กด **เปิดเส้นทางใน Google Maps** — นำทางครบทุกจุดจากตำแหน่งปัจจุบัน
4. **ลูกค้าเปลี่ยนใจกลางทาง?** พิมพ์/พูดความต้องการใหม่ → AI แนะนำทรัพย์ที่ตรงกว่า + ชี้จุดเดิมที่ไม่ตรงแล้ว → กดเพิ่มเข้ารูทได้ทันที
5. **บนแอปมือถือ:** เช้าก่อนวันนัดจะมีแจ้งเตือนเด้งเตือนแผนของพรุ่งนี้ให้อัตโนมัติ

---

## 5. ทำเอกสารเปรียบเทียบเสนอลูกค้า + พิมพ์ PDF

1. เข้าหน้า **เปรียบเทียบ** (จากหน้ารายการ หรือสั่งผู้ช่วย AI ว่า "เปิดหน้าเปรียบเทียบ")
2. พิมพ์รหัสทรัพย์เพื่อเพิ่มเข้าเทียบ **2–4 รายการ**
3. ใส่ชื่อลูกค้า/ความต้องการ → AI เขียนบทวิเคราะห์ จุดเด่น/ข้อควรพิจารณา/คำแนะนำ ให้
4. กด **พิมพ์ / บันทึก PDF** → ได้เอกสารหัวกระดาษ HUP แนวนอน ส่งลูกค้าทาง LINE ได้เลย
   - **บนแอป:** หน้าต่างพิมพ์ของเครื่องเด้งขึ้น เลือกเครื่องพิมพ์ หรือ **"บันทึกเป็น PDF"**

---

## 6. ผู้ช่วย AI (ปุ่ม ✨ ทุกหน้า)

- **ถามข้อมูลเป็นภาษาพูด** เช่น *"โกดังแถวบางพลีงบไม่เกินแสนมีไหม"* → ตอบจากข้อมูลจริง พร้อมการ์ดกดดูต่อ (ไม่มีก็บอกตรงๆ ไม่มโน)
- **สั่งงานผ่านแชท** เช่น "สร้างแผนเยี่ยมชมให้ลูกค้า A", "เพิ่มทรัพย์ WH-012 เข้ารูท" — ทุกคำสั่งมีปุ่มยืนยันก่อนทำจริง
- กด **ไมค์** พูดถามได้ + เปิดโหมด **อ่านออกเสียง** ตอนขับรถ

---

## 7. ดูภาพรวมพอร์ต (สรุป)

- มูลค่าเช่า/ขายรวม, ทรัพย์ใหม่เดือนนี้, นัดที่ใกล้ถึง
- แตะกราฟสัดส่วนประเภท/ทำเล → เปิดหน้ารายการพร้อมกรองให้
- **สุขภาพข้อมูล:** ชี้ทรัพย์ที่ยังไม่มีรูป/พิกัด/ราคา/เบอร์ หรือค้างเก่าเกิน 90 วัน — กดเข้าไปเติมได้
- **AI วิเคราะห์พอร์ตรายวัน:** จุดแข็ง, ความต้องการที่ยังไม่มีของ, ทรัพย์ที่ควรเร่งนำเสนอ

## สำหรับแอดมินทีม

### เพิ่มสมาชิกลูกทีม
1. กดเมนู **ทีม**
2. **เพิ่มสมาชิก** ใส่อีเมล → สมาชิกใหม่ล็อกอินด้วยอีเมลนั้นได้เลย
3. ดูรายชื่อ/บทบาท และเปลี่ยนชื่อองค์กรได้ที่หน้านี้

### ดูประวัติการใช้งาน
- เห็นได้เฉพาะแอดมิน: ใครเพิ่ม/แก้/ลบทรัพย์ · สร้างแผน · นำเข้าไฟล์ · ใช้ AI เมื่อไหร่
- กรองตามประเภท/ค้นหาชื่อคนได้ (บันทึกถาวร แก้/ลบย้อนหลังไม่ได้)

---

## แอปมือถือ

### ติดตั้งครั้งแรก (Android)
- เปิดลิงก์ APK ที่แอดมินส่งให้ → แตะติดตั้ง → ครั้งแรกกดอนุญาต "ติดตั้งจากแหล่งที่ไม่รู้จัก"
- หรือติดตั้งแบบ PWA: เปิดเว็บใน Chrome → เมนู → **เพิ่มไปที่หน้าจอหลัก**

### การอัปเดต
- ส่วนใหญ่แอป **อัปเดตเอง** เมื่อเปิดครั้งถัดไป — ไม่ต้องทำอะไร
- ถ้ามีแถบ **"แอปเวอร์ชันใหม่พร้อมติดตั้ง"** เด้งขึ้น → แตะ **ดาวน์โหลด** แล้วติดตั้งทับ

### สิทธิ์ที่แอปขอ (กดอนุญาตเมื่อใช้ครั้งแรก)
- **กล้อง** — ถ่ายรูปทรัพย์ · **ตำแหน่ง** — ปุ่มตำแหน่งฉัน · **แจ้งเตือน** — เตือนแผนเยี่ยมชม

## แก้ปัญหาเบื้องต้น

| อาการ | วิธีแก้ |
|---|---|
| ล็อกอินไม่ได้ | เช็คอีเมล/รหัสให้ตรง — ยังไม่ได้ ให้แอดมินรีเซ็ตให้ |
| ไม่เห็นปุ่มพูด | อุปกรณ์/เบราว์เซอร์ไม่รองรับ ให้พิมพ์แทน (Chrome รองรับดีสุด) |
| ปุ่มตำแหน่ง/กล้องกดแล้วไม่ทำงาน | ยังไม่ได้อนุญาตสิทธิ์ — เปิดสิทธิ์ให้แอปในตั้งค่าเครื่อง |
| ข้อมูลไม่อัปเดต | ดึงรีเฟรช หรือปิด-เปิดแอปใหม่ |
| AI ตอบช้า/ไม่ตอบ | เช็คอินเทอร์เน็ต แล้วลองใหม่อีกครั้ง |

ติดปัญหาที่แก้เองไม่ได้ → ติดต่อแอดมินทีม หรือทีมงาน HUP
