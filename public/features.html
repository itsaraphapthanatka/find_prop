HUP

ระบบจัดการทรัพย์ให้เช่า/ขายสำหรับทีมนายหน้า — โกดัง โรงงาน โชว์รูม ออฟฟิศ ครัวกลาง เว็บแอปทดแทน AppSheet เดิมทั้งหมด · แอปมือถือ native ทั้ง Android และ iOS จากโค้ดชุดเดียว · มี AI ผู้ช่วยในทุกขั้นตอนงาน

iOS + Androidแอป native โค้ดชุดเดียว
~50ฟิลด์ข้อมูลต่อทรัพย์
128รับคำขอ AI พร้อมกันสูงสุด

01

รายการทรัพย์ หน้าหลักของทีม

  • ค้นหาจากทุกฟิลด์ + ตัวกรอง ประเภททรัพย์ / เช่า-ขาย / จังหวัด / ช่วงราคา
  • มือถือแสดงเป็นการ์ดอ่านง่าย จอใหญ่เป็นตาราง — responsive ทุกขนาดจอ
  • รายละเอียดครบทุกฟิลด์ พร้อมรูป แผนที่ และปุ่มลัด โทร / SMS / LINE / Google Maps
  • รองรับ deep-link ตัวกรอง — จิ้มกราฟจาก Dashboard แล้วเปิดรายการพร้อมกรองให้ทันที
02

ฟอร์มเพิ่ม/แก้ไขทรัพย์

  • ครบ ~50 ฟิลด์ตามต้นแบบ — ราคา พื้นที่ ไฟฟ้า พื้นรับน้ำหนัก โซนสี ข้อมูลเจ้าของ ฯลฯ
  • Dropdown แบบ "เลือกหรือพิมพ์เพิ่ม" + multi-select คุณสมบัติเด่น / การใช้งาน / โซน
  • อัปโหลดรูปหลายรูปขึ้นคลังภาพ (Supabase Storage)
  • ปักพิกัดตำแหน่งทรัพย์จากแผนที่ในฟอร์มได้โดยตรง
03

บันทึกทรัพย์ด้วยเสียง AI

  • กดพูดหน้างาน เล่ารายละเอียดทรัพย์รวดเดียว → AI แกะเป็นฟิลด์ กรอกฟอร์มให้อัตโนมัติ
  • เข้าใจตัวเลขที่พูดเป็นคำไทย เช่น "ค่าเช่าแปดหมื่นห้า" → 85,000
  • แยกข้อมูลลงถูกช่อง — จุดเด่นเข้า "คุณสมบัติ" ไม่ปนหมายเหตุ และไม่แต่งข้อมูลที่ไม่ได้พูด
  • ถอดเสียงในเบราว์เซอร์ (ฟรี ไม่มีค่าบริการ) — เครื่องที่ไม่รองรับจะซ่อนปุ่มพูดให้เอง
04

แผนที่รวมทรัพย์

  • ปักหมุดทรัพย์ทั้งหมดที่มีพิกัดบนแผนที่เดียว พร้อม popup สรุปย่อ + เปิดรายละเอียดเต็ม
  • หมุดแยกตามประเภท — สีประจำประเภท + ไอคอนในหัวหมุด (โรงงาน/โชว์รูม/โกดัง/ออฟฟิศ/ครัวกลาง) ใช้สีคู่กับรูปเสมอ อ่านออกแม้ตาบอดสี
  • แถบสี (legend) ล่างแผนที่ บอกสี=ประเภท + จำนวน แตะกรองดูเฉพาะประเภท มีชิพ "ทั้งหมด" กดกลับ
  • ปุ่ม ตำแหน่งฉัน กระโดดไปจุดปัจจุบัน (บนแอปใช้ GPS ของเครื่องแบบ native)
  • หมุดร่างสีม่วงสำหรับกำหนดตำแหน่งทรัพย์ใหม่ก่อนบันทึก
05

แผนเยี่ยมชม / จัดรูทพาลูกค้า AI

  • สร้างแผนต่อลูกค้า เลือกทรัพย์เป็นจุดแวะ จัดลำดับก่อน-หลังได้อิสระ
  • เปิดเส้นทางใน Google Maps ครบทุกจุดแวะในคลิกเดียว — นำทางเริ่มจากตำแหน่งปัจจุบัน
  • แจ้งเตือน push บนมือถือ — ทุกเช้าเตือนสมาชิกที่มีแผนเยี่ยมชมของวันพรุ่งนี้ แตะเปิดหน้าแผนได้ทันที
  • AI จับคู่ requirement ใหม่: ลูกค้าเปลี่ยนความต้องการกลางทาง (พิมพ์หรือพูด) → AI สแกนทรัพย์ทั้งระบบ แนะนำตัวที่ตรง พร้อมชี้จุดแวะเดิมที่ไม่ตรงแล้ว → กดเพิ่มเข้ารูทได้ทันที
06

ชอร์ตลิสต์เปรียบเทียบ เอกสารเสนอลูกค้า AI

  • เลือกทรัพย์ 2–4 รายการ เทียบสเปกเป็นตารางเรียงข้างครบทุกหัวข้อสำคัญ
  • AI เขียนบทวิเคราะห์ จุดเด่น / ข้อควรพิจารณา / คำแนะนำ ประกอบการตัดสินใจ
  • พิมพ์ / บันทึก PDF เอกสารหัวกระดาษ HUP (แนวนอน A4) — บนแอปสั่งผ่านระบบพิมพ์ของเครื่องได้เลย
  • เปิดจากหน้ารายการ หรือสั่งผ่านผู้ช่วย AI ก็ได้
07

นำเข้าข้อมูลจาก Excel / CSV

  • 3 ขั้นตอน: อัปโหลดไฟล์ → ตรวจ/จับคู่คอลัมน์ → นำเข้า (รองรับไฟล์จาก AppSheet / Google Sheets เดิม)
  • จับคู่หัวคอลัมน์กับฟิลด์อัตโนมัติ รู้จักชื่อภาษาไทยหลายแบบ และแก้เองได้ก่อนนำเข้า
  • แปลงวันที่ พ.ศ. / วันที่ Excel / ตัวเลขมีคอมมา ให้ถูกต้องอัตโนมัติ
  • ตรวจรหัสทรัพย์ซ้ำ เลือก "ข้าม" หรือ "อัปเดตทับ" + มีเทมเพลต CSV ให้ดาวน์โหลด
08

Dashboard สรุปภาพรวม AI

  • ตัวเลขพอร์ต: มูลค่าเช่า/ขายรวม ทรัพย์ใหม่เดือนนี้ + กราฟย้อนหลัง 6 เดือน
  • นัดเยี่ยมชมที่ใกล้ถึง พร้อมปุ่มนำทางไปตามรูท
  • สัดส่วนประเภททรัพย์ / เช่า-ขาย — แตะกราฟเพื่อเปิดหน้ารายการพร้อมตัวกรอง
  • ทำเลยอดนิยม + ค่าเช่าเฉลี่ยต่อ ตร.ม. แยกประเภท พร้อมชี้ทรัพย์ที่ถูก/แพงกว่ากลุ่ม
  • สุขภาพข้อมูล: ทรัพย์ไม่มีรูป / พิกัด / ราคา / เบอร์ติดต่อ / ค้างเก่าเกิน 90 วัน
  • AI วิเคราะห์พอร์ตรายวัน — จุดแข็ง, demand ที่ยังไม่มีของ, ทรัพย์ควรเร่งนำเสนอ
09

ผู้ช่วย AI แชทลอย เรียกได้ทุกหน้า AI

  • ถามข้อมูลทรัพย์/แผนเยี่ยมชมของทีมเป็นภาษาพูด เช่น "โกดังแถวบางพลีงบไม่เกินแสนมีไหม" → ตอบจากข้อมูลจริงพร้อมการ์ดทรัพย์กดดูต่อ
  • ไม่มีข้อมูลก็ตอบตรงๆ — ออกแบบให้ไม่มโนข้อมูลที่ไม่มีในระบบ
  • สั่งงานผ่านแชท: เพิ่ม/ถอดทรัพย์จากรูท สร้างแผนเยี่ยมชม เปิดหน้าเปรียบเทียบ — ทุกคำสั่งมีปุ่มยืนยันก่อนทำจริง
  • พูดถามด้วยไมค์ + โหมดอ่านคำตอบออกเสียง (เหมาะตอนขับรถ) และจำบทสนทนาเดิมไว้
10

ทีมและสิทธิ์การใช้งาน

  • ล็อกอินด้วยอีเมล — ข้อมูลแยกตามองค์กรด้วย Row Level Security ทั้งระบบ
  • หน้าทีม (แอดมินองค์กร): เพิ่มสมาชิกลูกทีม ดูรายชื่อ/บทบาท เปลี่ยนชื่อองค์กร
  • หน้า Super Admin (เจ้าของระบบ): เห็นทุกองค์กร จำนวนสมาชิก/ทรัพย์ ตั้งแพ็กเกจ free / pro / enterprise และวันหมดอายุ
  • มุมมอง super โหมดภาพรวม: ป้ายชื่อองค์กร + dropdown กรององค์กรครบทุกหน้า (รายการ/แผนที่/สรุปภาพรวม/แผนเยี่ยมชม/Super Admin) — Dashboard และ AI insight คำนวณตามองค์กรที่เลือก
  • super สร้าง/แก้ทรัพย์แทนลูกค้าจากโหมดภาพรวม: ฟอร์มมีช่อง "องค์กรเจ้าของทรัพย์" บังคับเลือกก่อนบันทึก
  • สวมสิทธิ์องค์กร: super คลิกชื่อองค์กรเพื่อทำงานเสมือนสมาชิกองค์กรนั้น (เพิ่ม/แก้/นำเข้า) พร้อมแถบเตือน + ปุ่มออกจากสิทธิ์ตลอดเวลา
  • ประวัติการใช้งาน (เฉพาะแอดมิน): บันทึกอัตโนมัติว่าใครทำอะไรเมื่อไหร่ — ทรัพย์ / แผนเยี่ยมชม / นำเข้าไฟล์ / การใช้ AI — สมาชิกทั่วไปเปิดดูไม่ได้ และแก้/ลบย้อนหลังไม่ได้
11

แอปมือถือ native Android + iOS โค้ดชุดเดียว

  • แอปจริงลง Store ได้ — ห่อด้วย Capacitor จากโค้ดเว็บชุดเดียว ออกได้ทั้ง Android และ iOS
  • กล้อง ถ่ายรูปทรัพย์หน้างาน (ย่อขนาดก่อนอัปให้อัตโนมัติ) · GPS native ปุ่มตำแหน่งฉัน · push เตือนแผนเยี่ยมชม
  • พิมพ์ / บันทึก PDF เอกสารเปรียบเทียบผ่านระบบพิมพ์ของเครื่อง
  • อัปเดตเองไม่ต้องโหลดแอปใหม่: แก้ฝั่งเว็บแล้วแอปตามอัตโนมัติ (live-update) — งานที่แตะโค้ด native จะเด้งแถบชวนอัปเดตในแอป (iOS อัปผ่าน App Store/TestFlight)
  • แจก APK ตรงได้ ผ่านลิงก์ถาวรบน GitHub Releases — สร้างอัตโนมัติทุกครั้งที่ push
  • ยังติดตั้งเป็น PWA จากเบราว์เซอร์ได้เหมือนเดิม ใช้งาน offline ได้ — mobile-first, รองรับจอ edge-to-edge (Android 15 / Dynamic Island)
  • Landing page ขายแพลตฟอร์ม: คนทั่วไปเปิดเว็บเห็นหน้าขายระบบ HUP — ฟีเจอร์เด่น, แพ็กเกจ Free/Pro/Enterprise, ติดต่อทีมขาย — ลูกค้าปัจจุบันกด "เข้าสู่ระบบ"

สถาปัตยกรรมและความปลอดภัย

  • AI ทุกฟีเจอร์เรียกผ่าน proxy ฝั่งเซิร์ฟเวอร์ — คีย์ LLM ไม่อยู่ในหน้าเว็บ และตรวจการล็อกอินก่อนใช้ทุกครั้ง
  • LLM รันบนเครื่องของเราเอง (gemma-4-12b · vLLM · GPU ภายใน) — ข้อมูลทรัพย์ไม่ออกไปผู้ให้บริการภายนอก ไม่มีค่า AI รายเดือน
  • ก่อนส่งให้ AI ระบบคัดเฉพาะทรัพย์ที่เกี่ยวข้องกับคำถาม — ตอบแม่นและเร็วขึ้น
  • Row Level Security ทุกตาราง — แต่ละองค์กรเห็นเฉพาะข้อมูลของตัวเอง
  • แอปอัปเดตปลอดภัย — live-update กันดึงลงเวอร์ชันเก่า + bundle พังย้อนกลับเอง, APK เซ็นด้วยกุญแจกลางคงที่
  • push: device token ผูกผู้ใช้/องค์กร (RLS), ยิงจาก Vercel Cron ผ่าน FCM — คีย์อยู่ฝั่งเซิร์ฟเวอร์เท่านั้น
