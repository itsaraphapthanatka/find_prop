01 เพิ่มทรัพย์ใหม่

เลือกวิธีที่เหมาะกับสถานการณ์:

วิธี A

พิมพ์กรอกฟอร์ม

ครบทุกฟิลด์ เหมาะตอนนั่งโต๊ะ

วิธี B · เร็วสุด

พูดให้ AI กรอก 🎤

เล่ารวดเดียวหน้างาน AI แกะให้

วิธี C

นำเข้า Excel/CSV

ย้ายข้อมูลเก่าทีเดียวทั้งไฟล์

  1. วิธี A: กด ฟอร์ม (หรือ + เพิ่มทรัพย์) → กรอกวันที่ รหัส ประเภท เช่า/ขาย ทำเล ราคา — ช่องมี * คือบังคับ → บันทึก
  2. วิธี B: กดปุ่ม พูด แล้วเล่ารวดเดียว เช่น "โกดังบางพลี ห้าร้อยตร.ม. ค่าเช่าแปดหมื่นห้า สูงแปดเมตร" → AI กรอกให้ → ตรวจก่อนบันทึกทุกครั้ง
  3. วิธี C: เข้าหน้า นำเข้าข้อมูล → อัปโหลดไฟล์ → ระบบจับคู่คอลัมน์ให้ (แก้เองได้) → นำเข้า (รหัสซ้ำเลือกข้าม/อัปเดตทับ)
  4. แนบรูป: บนแอปกด ถ่ายรูปด้วยกล้อง · บนคอมกดกรอบรูปเลือกไฟล์
  5. ปักตำแหน่ง: แตะแผนที่เล็กในฟอร์ม หรือปักจากหน้าแผนที่
💡 ปุ่มพูดแสดงบน Chrome (คอม/Android) ถ้าไม่เห็นปุ่ม = อุปกรณ์นั้นไม่รองรับ ให้พิมพ์แทน · พูดตัวเลขไทยได้ "แปดหมื่นห้า" → 85,000

02 ค้นหาและกรองทรัพย์

  1. กด รายการ
  2. พิมพ์ในช่องค้นหา — ค้นได้จากรหัส ทำเล ประเภท ฯลฯ
  3. กรองเพิ่ม: ประเภท · เช่า/ขาย · จังหวัด · ช่วงราคา
  4. แตะการ์ด/แถวเพื่อดูรายละเอียดเต็ม + ปุ่มลัด โทร / SMS / LINE / Google Maps

03 ดูทรัพย์บนแผนที่

  • สีหมุด = ประเภททรัพย์ ดูคำอธิบายสีที่แถบล่างแผนที่ (legend)
  • แตะชิพประเภทที่ legend เพื่อ ดูเฉพาะประเภทนั้น · แตะ ทั้งหมด เพื่อกลับ
  • ปุ่ม ⊙ ตำแหน่งฉัน (มุมล่างขวา) กระโดดไปจุดที่คุณอยู่
  • เพิ่มทรัพย์จากแผนที่: วางหมุดเพิ่มทรัพย์ → แตะตำแหน่ง → กรอกฟอร์มต่อ

04 แผนพาลูกค้าดูทรัพย์

  1. กด แผนเยี่ยม → สร้างแผน
  2. ใส่ชื่อลูกค้า/ความต้องการ เลือกทรัพย์เป็นจุดแวะ จัดลำดับก่อน-หลังได้
  3. กด เปิดเส้นทางใน Google Maps — นำทางครบทุกจุดจากตำแหน่งปัจจุบัน
  4. ลูกค้าเปลี่ยนใจกลางทาง? พิมพ์/พูดความต้องการใหม่ → AI แนะนำทรัพย์ที่ตรงกว่า + ชี้จุดเดิมที่ไม่ตรง → กดเพิ่มเข้ารูทได้ทันที
🔔 บนแอปมือถือ เช้าก่อนวันนัดจะมี แจ้งเตือน เด้งเตือนแผนของพรุ่งนี้ให้อัตโนมัติ

05 เอกสารเปรียบเทียบเสนอลูกค้า + PDF

  1. เข้าหน้า เปรียบเทียบ (จากหน้ารายการ หรือสั่ง AI ว่า "เปิดหน้าเปรียบเทียบ")
  2. พิมพ์รหัสทรัพย์เพื่อเพิ่มเข้าเทียบ 2–4 รายการ
  3. ใส่ชื่อลูกค้า/ความต้องการ → AI เขียนบทวิเคราะห์ จุดเด่น/ข้อควรพิจารณา/คำแนะนำ ให้
  4. กด พิมพ์ / บันทึก PDF → เอกสารหัวกระดาษ HUP แนวนอน — บนแอปเลือก "บันทึกเป็น PDF" ส่ง LINE ให้ลูกค้าได้เลย

06 ผู้ช่วย AI (ปุ่ม ✨ ทุกหน้า)

  • ถามเป็นภาษาพูด เช่น "โกดังแถวบางพลีงบไม่เกินแสนมีไหม" → ตอบจากข้อมูลจริง พร้อมการ์ดกดดูต่อ (ไม่มีก็บอกตรงๆ ไม่มโน)
  • สั่งงานผ่านแชท เช่น "สร้างแผนเยี่ยมชมให้ลูกค้า A" — ทุกคำสั่งมีปุ่มยืนยันก่อนทำจริง
  • กด ไมค์ พูดถามได้ + เปิดโหมด อ่านออกเสียง ตอนขับรถ

07 ภาพรวมพอร์ต (สรุป)

  • มูลค่าเช่า/ขายรวม, ทรัพย์ใหม่เดือนนี้, นัดที่ใกล้ถึง
  • แตะกราฟสัดส่วนประเภท/ทำเล → เปิดหน้ารายการพร้อมกรองให้
  • สุขภาพข้อมูล: ชี้ทรัพย์ที่ยังไม่มีรูป/พิกัด/ราคา/เบอร์ หรือค้างเก่าเกิน 90 วัน → กดเข้าไปเติม
  • AI วิเคราะห์พอร์ตรายวัน: จุดแข็ง, ความต้องการที่ยังไม่มีของ, ทรัพย์ที่ควรเร่งนำเสนอ
สำหรับแอดมินทีม

08 จัดการทีม

  1. กดเมนู ทีม → เพิ่มสมาชิก ใส่อีเมล → สมาชิกใหม่ล็อกอินด้วยอีเมลนั้นได้เลย
  2. ดูรายชื่อ/บทบาท และเปลี่ยนชื่อองค์กรได้ที่หน้านี้
  3. ประวัติการใช้งาน: เห็นเฉพาะแอดมิน — ใครเพิ่ม/แก้/ลบทรัพย์ สร้างแผน นำเข้าไฟล์ ใช้ AI เมื่อไหร่ (บันทึกถาวร แก้ย้อนหลังไม่ได้)

09 แอปมือถือ

  • ติดตั้ง (Android): เปิดลิงก์ APK ที่แอดมินส่ง → ติดตั้ง → ครั้งแรกอนุญาต "ติดตั้งจากแหล่งที่ไม่รู้จัก" · หรือ PWA: Chrome → เมนู → เพิ่มไปหน้าจอหลัก
  • อัปเดต: ส่วนใหญ่แอปอัปเดตเองเมื่อเปิดครั้งถัดไป — ถ้ามีแถบ "เวอร์ชันใหม่พร้อมติดตั้ง" เด้ง ให้แตะ ดาวน์โหลด แล้วติดตั้งทับ
  • สิทธิ์ที่แอปขอ: กล้อง (ถ่ายรูป) · ตำแหน่ง (ปุ่มตำแหน่งฉัน) · แจ้งเตือน (เตือนแผนเยี่ยม) — กดอนุญาตเมื่อใช้ครั้งแรก

10 แก้ปัญหาเบื้องต้น

อาการวิธีแก้
ล็อกอินไม่ได้เช็คอีเมล/รหัสให้ตรง — ยังไม่ได้ให้แอดมินรีเซ็ตให้
ไม่เห็นปุ่มพูดอุปกรณ์/เบราว์เซอร์ไม่รองรับ ให้พิมพ์แทน (Chrome รองรับดีสุด)
ปุ่มตำแหน่ง/กล้องไม่ทำงานยังไม่อนุญาตสิทธิ์ — เปิดสิทธิ์ให้แอปในตั้งค่าเครื่อง
ข้อมูลไม่อัปเดตดึงรีเฟรช หรือปิด-เปิดแอปใหม่
AI ตอบช้า/ไม่ตอบเช็คอินเทอร์เน็ตแล้วลองใหม่
